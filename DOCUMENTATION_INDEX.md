## 💡 Tips for Success

### 1. Always Start with QUICK_REFERENCE
Before coding, check if your pattern exists in [`QUICK_REFERENCE.md`](./QUICK_REFERENCE.md)

### 2. Use Color Tokens
Always use semantic colors: `bg-accent`, `text-foreground` instead of `bg-[#C5D82D]`

### 3. Follow Width Pattern
All major containers: `w-[90%] md:w-[95%] mx-auto`

### 4. Responsive First
Write mobile styles first, then use `sm:`, `md:`, `lg:` prefixes

### 5. 300ms Transitions
Keep animations consistent with `transition-all duration-300`

### 6. Check Accessibility
Verify color contrast and keyboard navigation

### 7. Test Responsiveness
Test on mobile (375px), tablet (768px), desktop (1024px+)
